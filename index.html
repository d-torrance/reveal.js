M2 and reveal.js

Goal: Include syntax-highlighted Macaulay2 code in reveal.js slides:

			      
				-- the twisted cubic
				R = QQ[x,y,z,w]
				I = monomialCurveIdeal(R, {1, 2, 3})

Quick start

  1. Download and unzip reveal.js.
  2. Replace the file plugin/highlight/highlight.js with this version to add Macaulay2 support.
  3. Include your Macaulay2 code!
